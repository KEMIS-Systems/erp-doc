# Inventario

> Gerencie todos os inventarios.

#### Formulário

| Campo              | Obrigatório | Descrição                        |
| ------------------ | :---------: | -------------------------------- |
| Empresa            |   **Sim**   | Empresa responsável pelo estoque |
| Tipo de inventario |   **Sim**   | Tipo de inventário implementado  |
| Data               |   **Sim**   | Data                             |
| Concluido          |   **Sim**   | Status de conclusão              |

| Ícone                                  | Ação                                                                                        |
| -------------------------------------- | ------------------------------------------------------------------------------------------- |
